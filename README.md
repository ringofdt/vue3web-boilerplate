## usage

1. Prevent sync node_module to icloud:

    ```sh
    mkdir node_modules.nosync
    ln -svf node_modules.nosync node_modules
    ```

1. Install the packages:

   ```sh
   yarn install
   ```

1. Run

    ```sh
    yarn dev
    ```
